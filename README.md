## 中英对照

| 中文                 | 英文
|:---------------------|:-------------------------------------
| 【必须】             | MUST
| 【禁止】             | MUST NOT
| 【应当】             | SHOULD
| 【不应】             | SHOULD NOT
| 【可能】             | MAY
| 【可选的】           | OPTIONAL
|                      |
| 前提性引用           | Normative References
| 标准跟踪文档         | standards track documents
|                      |
| 互联网数字分配机构   | IANA, Internet Assigned Numbers Authority
| 国际互联网工程任务组 | IETF, Internet Engineering Task Force
